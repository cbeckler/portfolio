xlxswriter Automated Formatting Functions

Charlie developed a comprehensive suite of functions to automate report formatting in Excel when exporting pandas dataframes via the xlxswriter package. These functions create nice-looking reports with formatting responsive to changes in data size or shape. There are default aesthetic options but customization is possible with optional arguments in most functions.

The functions are capable of formatting one-dimensional, two-dimensional, three-dimensional, and four-dimensional data, with full capabilty to hand multiindices for rows, columns, or both at once. The automation is positionally based and largely reliant on matrix math and looping, with additonal functionality and formatting via conditionals.

pandas Automated Analysis

Charlie is developing a comprehensive suite of functions to automate the pandas data analysis and reshaping and labeling results to be report-ready. These functions will take in raw data and perform specified analyses and then output a clean dataframe ready to be exported. Customization of labeling and ordering of data is possible via optional arguments in most functions.

The functions are capable of doing simple groupby analyses, or groupby analyses that result in multiindices for rows, columns, or both. It is also capable of pivoting dataframes. The automation is done via a software engineering persepective, using the given parameters to process the data from raw to finished form.

This package was developed to be fully compatible with the xlsxwriter Automated Formatting Functions, and will result in dataframes that can be immediately exported to Excel and formatted with those functions.

Gender Dysphoria Factor Analysis

Charlie collected his own data and performed an exploratory factor analysis and structural equation modeling in R on gender dysphoria to investigate whether it presents differently between transgender man and transgender women. Charlie was able to produce a validated, alternative model for analyzing gender dysphoria in transgender individuals.

This project was initially completed as part of the USC Honors Thesis program, with Charlie given special permission to not work directly with any pre-existing lab. Charlie won first place for Social Sciences at the Senior Research Symposium in 2019 for this project.

Charlie is currently pursuing publication of this research.
